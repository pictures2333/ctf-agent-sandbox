# Rules

You should always follow the rules.

## General

- You should always read the following files before you start to work:
    - This file (AGENTS.md)
    - All files about your task.
- Use ``MCP Terminal`` to execute command when:
    - You need to use interactive tools (For example: GDB)
    - You need to interact with the target program.
- You can use ``web_search`` for more info:
    - writeups
    - payloads
    - and more...

## Stop Condition

Do not stop until you reach one of the following conditions:

- You get the flag.
- The environment is broken.
- You give up.
